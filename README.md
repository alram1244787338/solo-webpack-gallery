# 图片画廊 · Image Gallery

一个基于原生 JavaScript 的响应式图片画廊应用，支持瀑布流展示、标签筛选、排序切换、关键词搜索等功能。

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器 (http://localhost:8081)
npm run dev

# 生产环境构建
npm run build

# 运行单元测试
npm test
```

## 功能特性

- **瀑布流布局**：基于 CSS `columns` 实现，图片按高度自适应排列
- **标签筛选**：5 种预设标签（全部 / 风景 / 人物 / 动物 / 建筑 / 美食），点击即时过滤
- **排序切换**：
  - 最新上传：按 `uploadDate` 降序
  - 随机浏览：Fisher-Yates 洗牌算法随机重排，切回最新上传可正确恢复顺序
- **关键词搜索**：按图片标题搜索，大小写不敏感，带 200ms 防抖
- **图片交互**：卡片 hover 抬升 + 渐变遮罩显示标题和日期
- **响应式适配**：根据屏幕宽度自动调整列数（4 / 3 / 2 / 1 列）

| 类别 | 技术 | 说明 |
|------|------|------|
| 语言 | 原生 JavaScript (ES6+) | Class-based OOP 结构 |
